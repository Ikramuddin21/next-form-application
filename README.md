## Overview

This project form submission & data showing.

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- Ant Design

## Installation

Clone the repository:

```bash
git clone https://github.com/Ikramuddin21/next-form-application
cd next-form-application
npm install
npm run dev
```
